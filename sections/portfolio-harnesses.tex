\subsec{Autonomous Agent Harnesses \& Loops}
\begin{itemize}
  \item Self-healing multi-agent orchestrator: a SQLite priority queue, multi-consumer leases, parallel workers, and dispatch/fan-out for agent-team parallelism.
  \item Watchdog recovery for parked and coasting agents --- detecting errors, stalls, and idle-while-work-pending, then recovering agents or enforcing stay-busy execution at the tooling layer.
  \item Reliable inter-agent messaging around tooling failure modes; continuous resolve-or-decompose issue loops.
  \item Agents doing real autonomous work: credentialed document retrieval + workflow prep, risk-capped live-capital market-making, CI repair, repo changes, long-running task decomposition.
\end{itemize}
