\subsec{Agent Skills, MCP \& Tooling}
\begin{itemize}
  \item Practical agent-architecture framework: MCP servers expose callable capabilities; skills encode reusable judgment, procedures, and operating taste.
  \item Public + internal agent-skills/MCP authoring workflows --- including a skill for when to use MCP vs skills, privacy scanning, security-audit gates, and broker-held credential boundaries.
  \item Agents that build their own MCP servers and contribute reusable skills back, with review gates for privacy, security, and credential exposure.
  \item Agent-facing tool/skill layers for capability selection, bounded execution, stateful workflows, and low-noise operation.
\end{itemize}
